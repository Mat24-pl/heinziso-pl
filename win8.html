Windows 8

Wersja: Windows 8 Pro
Data premiery: 26 Października 2012
Architektura: 64-bit / 32-bit

Windows 8 wprowadził nowy interfejs Modern UI (Metro), który miał pasować do ekranów dotykowych... Problem w tym, że większość użytkowników miała myszkę i klawiaturę 😅. Brak przycisku Start i całkowicie nowa nawigacja sprawiły, że wielu czuło się jak na obcej planecie.

🔧 Wymagania sprzętowe

  • Procesor: 1 GHz z obsługą PAE, NX i SSE2
  • Pamięć RAM: 1 GB (32-bit) lub 2 GB (64-bit)
  • Dysk twardy: 16 GB (32-bit) lub 20 GB (64-bit)
  • Karta graficzna: Microsoft DirectX 9 z WDDM

💽 Instrukcja instalacji

  1. Pobierz obraz ISO poniżej.
  2. Stwórz bootowalny pendrive (np. z użyciem Rufusa).
  3. Uruchom komputer z USB i postępuj zgodnie z kreatorem instalacji.
  4. Ustaw konto Microsoft (albo i nie 😉) i odkrywaj świat kafelków!
